# QuickDmail — Duck 邮箱接码助手

> Chrome 扩展 · Manifest V3 · 版本 1.5.0

自动捕获 Gmail 中的 HeyGen 验证链接，配合 DuckDuckGo 私密邮箱批量生成，让**生成地址 → 注册账号 → 接收验证码 → 一键验证**的全流程在一个弹窗内完成。

## 功能概览

| 功能 | 说明 |
|------|------|
| **一键生成 Duck 地址** | 调用 DDG API 即时生成 `xxx@duck.com` 匿名邮箱，无需打开任何网页 |
| **自动捕获验证链接** | 实时监听 Gmail，邮件到达即提取 HeyGen 激活链接 |
| **链接绿光指示** | 验证邮件到达后，对应 Duck 地址行的「链接」按钮自动**变绿发光** |
| **一键复制** | 点击绿光按钮，验证链接直接写入剪贴板 |
| **记录归档** | 所有捕获的链接和注册账号自动归档，支持搜索和 CSV 导出 |
| **Google Sheets 同步** | 一次 OAuth 授权后，历史数据自动备份到 Google Sheets |

## 安装方法

> 本插件为本地开发版，需手动加载，不在 Chrome 应用商店发布。

**第一步：下载代码**

```bash
git clone https://github.com/stevenpcny/QuickDmail.git
```

或点击页面右上角 **Code → Download ZIP**，解压到本地任意位置。

**第二步：打开扩展管理页**

在 Chrome 地址栏输入：

```
chrome://extensions
```

**第三步：开启开发者模式**

点击右上角的「**开发者模式**」开关，将其打开（蓝色状态）。

**第四步：加载插件**

点击「**加载已解压的扩展程序**」→ 选择刚才下载/解压的项目文件夹（包含 `manifest.json` 的那一层）。

**第五步：打开 Gmail**

在 Chrome 中打开 [mail.google.com](https://mail.google.com)，工具栏会出现插件图标，点击即可打开弹窗。

> **提示**：插件必须在 Gmail 标签页已打开的情况下使用，首次打开 Gmail 后插件会自动开始监控。

## 首次配置

### 1. 配置 DuckDuckGo Token

Token 用于直接调用 DDG API 生成新地址，**配置一次即可长期使用**（token 有效期较长）。

点击右上角 **⚙** 进入设置，找到「**DuckDuckGo Token**」区块：

**获取 Token 的步骤：**

1. 在 Chrome 中打开：`duckduckgo.com/email/settings/autofill`
   （需要已登录 DuckDuckGo 账号）

2. 按 `F12` 打开开发者工具，切到 **Network（网络）** 标签

3. 点击页面上的「**Generate**」按钮生成一个新地址

4. 在 Network 面板中找到发往 `quack.duckduckgo.com` 的请求，点击它

5. 在右侧「**Headers（标头）**」中找到：
   ```
   Authorization: Bearer y44slhxl...
   ```
   复制 `Bearer ` 后面的那串字符（即 token 值）

6. 回到插件设置页，将 token 粘贴到输入框，点击「**保存**」

配置成功后会显示：`● Token: y44slhxl... · X 小时前保存`

---

### 2. 连接 Google 账号（可选）

连接 Google 账号后，插件可以：
- 通过 Gmail API 进行**全量历史扫描**（补捉已有的验证邮件）
- 自动将数据备份同步到 **Google Sheets**

在设置页找到「**GOOGLE 集成 · 需要 OAUTH 授权**」区块，点击「**连接 Google 账号**」，按照弹出的 OAuth 授权流程完成授权即可。

> 未连接 Google 账号时，插件仍可通过 XHR 拦截实时捕获验证链接，只是无法使用全量扫描和 Sheets 同步功能。

## 界面说明

### 主界面

打开插件后默认显示主界面，顶部工具栏：

```
● 监控中   + 生成新 Duck 地址   记录   打开 DDG   ⚙
```

| 按钮 | 功能 |
|------|------|
| `● 监控中`（绿色） | 当前正在监控 Gmail，点击后变为「停止监控」 |
| `+ 生成新 Duck 地址` | 调用 DDG API 生成一个新的 `@duck.com` 地址，自动置顶显示 |
| `记录` | 打开链接记录面板，查看所有捕获的验证链接 |
| `打开 DDG` | 在新标签页打开 DDG 邮箱设置页（手动生成地址时使用） |
| `⚙` | 打开设置面板 |

**当前地址区**

主界面中部蓝色框显示**最新生成的 Duck 地址**，右侧复制按钮可一键复制到剪贴板，用于填写注册表单。

**历史地址列表**

列表显示所有已生成的 Duck 地址，每一行包含：

| 列 | 说明 |
|----|------|
| 地址 | `xxx@duck.com` 匿名邮箱地址 |
| `生成` 按钮 | 以该行时间戳为起点，手动触发重新关联（一般无需点击） |
| 时间 | 该地址的生成时间 |
| **`链接` 按钮** | 默认灰色；当对应验证邮件到达时**自动变绿发光**，点击复制验证链接 |
| 复制按钮 | 复制该行 Duck 地址到剪贴板 |

> **链接匹配逻辑**：以某个 Duck 地址的生成时间为起点，到下一个 Duck 地址的生成时间为终点，这段时间内捕获到的验证链接自动归属到该地址行。因此**先生成地址、再去注册**，链接才能正确匹配。

---

### 记录面板

点击顶部「**记录**」按钮进入，包含两个子页：

#### 链接列表

显示所有已捕获的验证链接，操作栏：

| 按钮 | 功能 |
|------|------|
| `停止监控` | 暂停实时监控（再次点击变为「开始监控」） |
| `扫描当前页` | 立即扫描 Gmail 当前可见的邮件列表，补捉未自动捕获的链接 |
| `⋮` | 更多操作（全选、导出 CSV 等） |

搜索框支持按**账号邮箱或链接内容**过滤，右侧显示当前记录总数。

表格列说明：

| 列 | 说明 |
|----|------|
| 注册账号 | 收到验证邮件的 Duck 地址 |
| 验证链接 | 捕获的完整激活链接（`auth.heygen.com/magic-web/...`） |
| 收件时间 | 邮件到达时间，默认按此列倒序排列 |
| 操作 | 复制链接 / 在新标签页打开 |

> **粗体行**：表示该条链接尚未被复制或使用，属于未读状态。

#### 账号汇总

汇总所有出现过的注册邮箱，记录每个地址的首次收件时间、最近收件时间和接码次数，支持导出 CSV。

---

### 设置面板

点击顶部「**⚙**」进入，共三个区块：

#### DuckDuckGo Token

粘贴从 DDG Network 面板获取的 Bearer token，点击保存。已保存的 token 会显示前几位字符和保存时间，点击「移除」可清空。

#### Duck 历史数据

| 按钮 | 功能 |
|------|------|
| `同步 Sheets` | 立即将历史数据推送到 Google Sheets |
| `导出 CSV` | 将所有 Duck 地址和对应链接下载为 CSV 文件 |
| `清空历史` | **不可逆操作**，清空本地所有 Duck 地址和链接记录 |

#### Google 集成 · Sheets 备份同步

1. 点击「**连接 Google 账号**」完成 OAuth 授权
2. 在 `Spreadsheet` 输入框粘贴已有 Google Sheet 的 URL，或留空（首次同步时自动创建名为「Duck邮箱接码备份」的表格）
3. 勾选「**启用自动同步**」后，每 15 分钟及关闭 Gmail 时会自动同步一次
4. 点击「**立即同步 →**」手动触发，「**← 从 Sheets 导入**」可将 Sheets 数据拉回本地

## 完整使用流程

以注册一个 HeyGen 账号为例：

```
第 1 步  打开 Gmail，点击插件图标 → 确认顶部显示「● 监控中」

第 2 步  点击「+ 生成新 Duck 地址」→ 主界面顶部出现新地址
         例如：justify-line-relax@duck.com

第 3 步  点击地址旁的复制按钮，将地址复制到剪贴板

第 4 步  打开 HeyGen 注册页面，将 Duck 地址粘贴到邮箱输入框，提交注册

第 5 步  HeyGen 发出验证邮件 → DDG 自动转发到你的真实 Gmail
         插件后台实时拦截，无需手动操作

第 6 步  历史地址列表中，该 Duck 地址对应的「链接」按钮变绿发光

第 7 步  点击绿光「链接」按钮 → 验证链接自动复制到剪贴板

第 8 步  在浏览器地址栏粘贴并回车 → HeyGen 账号验证完成
```

> **等不到绿光？** 点击「记录」→「扫描当前页」，手动触发一次 Gmail 扫描。

## 进阶场景

### 批量注册多个账号

1. 重复点击「**+ 生成新 Duck 地址**」，每次注册前各生成一个新地址
2. 按顺序：生成地址 → 用该地址注册 → 等绿光 → 点击验证
3. 所有记录汇总在「**记录 → 链接列表**」中，按收件时间倒序排列

> **注意**：不要在注册完成前提前生成下一个地址，否则时间段会重叠导致链接匹配混乱。

### 数据备份到 Google Sheets

1. 设置页完成 Google OAuth 授权
2. 勾选「启用自动同步（每 15 分钟 + 关闭 Gmail 时）」
3. `Spreadsheet` 留空 → 点击「立即同步 →」→ 插件自动在你的 Google Drive 创建备份表格
4. 后续每次同步自动追加新记录，不覆盖旧数据

### 手动打开 DDG 页面生成地址

如果 token 失效或不想配置 token，可以使用手动模式：

1. 点击「**打开 DDG**」→ 在 DDG 设置页点击「Generate」
2. 点击「Copy」复制地址
3. 插件会自动检测到剪贴板写入，将新地址捕获并保存到历史列表

## 技术说明

| 文件 | 说明 |
|------|------|
| `manifest.json` | Manifest V3，声明权限与 content scripts |
| `content.js` | Gmail 页面注入脚本：DOM 监听、Gmail 已读标记、链接捕获 |
| `page-interceptor.js` | Gmail MAIN world 脚本：拦截 XHR/fetch，提取验证链接和邮件信息 |
| `ddg-page-interceptor.js` | DDG 页面 MAIN world 脚本：拦截剪贴板写入、DOM 轮询、fetch 捕获 Duck 地址 |
| `ddg-content.js` | DDG 页面 ISOLATED world 脚本：接收地址消息并存入 chrome.storage |
| `background.js` | Service Worker：消息路由、Google Sheets 同步、徽章更新、定时监控 |
| `popup.html/js` | 插件弹窗 UI：主界面、记录面板、设置面板全部功能 |

## 权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 保存捕获的链接、Duck 地址、配置信息 |
| `tabs` / `scripting` | 向 Gmail 页面注入脚本，执行已读标记和链接捕获 |
| `activeTab` | 获取当前标签页信息 |
| `notifications` | 新验证邮件到达时发送系统通知 |
| `identity` | Google OAuth 授权（Sheets 同步） |
| `alarms` | 定时监控任务（每 15 分钟自动同步） |
| `https://mail.google.com/*` | 监听和操作 Gmail 页面 |
| `https://sheets.googleapis.com/*` | 向 Google Sheets 写入备份数据 |
| `https://www.googleapis.com/*` | Google API 调用 |
| `https://quack.duckduckgo.com/*` | 调用 DDG API 生成 Duck 地址 |
| `https://duckduckgo.com/email/*` | 在 DDG 设置页注入捕获脚本 |
